The runtimes folder contains native binaries for PDFium (without V8 and XFA).
Compiled by and downloaded from https://github.com/bblanchon/pdfium-binaries.

The native libraries for SkiaSharp are taken and extracted from the matching NuGet packages:
https://www.nuget.org/packages/SkiaSharp.NativeAssets.Win32
https://www.nuget.org/packages/SkiaSharp.NativeAssets.Linux.NoDependencies
https://www.nuget.org/packages/SkiaSharp.NativeAssets.macOS
